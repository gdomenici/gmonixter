# Guido-Monixter Board Game

## Running & playing the game

- To start a dev server: `npm run dev` then navigate to http://localhost:5173/
- Input a YouTube playlist URL, such as `https://www.youtube.com/watch?v=t99KH0TR-J4&list=PLcMK01bSTnE65yJUpsD4E3kAyy4kLBt7M`
- Play!
